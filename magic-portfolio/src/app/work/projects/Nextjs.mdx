## Overview

A modern personal finance tracking application designed to help users visualize their income, expenses, and overall financial health through an intuitive and interactive interface.

## Key Features

- **Data Visualization**: Integrated dynamic charts and graphs using libraries like Recharts to provide clear visual insights into spending patterns and budget allocations.
- **Transaction Management**: Developed a comprehensive system for users to add, edit, and categorize financial transactions with real-time updates.
- **Secure Authentication**: Implemented user authentication to ensure personal financial data remains private and accessible only to the authorized user.
- **Responsive Dashboard**: Designed a mobile-friendly layout that provides a seamless experience across desktops, tablets, and smartphones using Tailwind CSS.

## Technologies Used

- **React & Vite**: For building a fast, component-based user interface with an optimized development environment.
- **Tailwind CSS**: Used for rapid styling and creating a clean, professional aesthetic.
- **Lucide React**: For consistent and modern iconography throughout the dashboard.
- **Vercel**: Leveraged for automated deployment and hosting.

## Challenges and Learnings

The main challenge was managing complex state transitions and ensuring that data visualizations updated instantly as user input changed. This project deepened my understanding of state management in React and how to effectively integrate third-party charting libraries with custom UI components. I also refined my skills in creating accessible data tables and forms for a better user experience.

## Outcome

he Finance Dashboard provides a functional and visually appealing tool for personal wealth management. It demonstrates my ability to handle data-heavy applications while maintaining high performance and a polished UI, serving as a key piece of my web development portfolio.
